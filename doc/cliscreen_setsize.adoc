== NAME
cliscreen_setsize - set the screen's size

== SYNOPSIS
[verse]
____
*#include "vulcalien/cliscreen.h"*

**void cliscreen_setsize(u32 **__w__**, u32 **__h__**);**
____

== DESCRIPTION
The *cliscreen_setsize*() function sets the size of the screen.

== RETURN VALUE
No value is returned.
